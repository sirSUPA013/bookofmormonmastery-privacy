Privacy Policy

Book of Mormon Mastery

Last Updated: December 28, 2025

The short version: You can use the app without an account—your progress stays on your device. If you create an optional account, your progress syncs securely to the cloud. We collect minimal anonymous analytics to improve the app.

Information We Collect

Information Stored Locally on Your Device

The App stores the following data locally on your device using your device's built-in storage:

  • Game Progress: Scores, achievements, streaks, and game history
  • User Preferences: Settings and customizations you choose
  • Study Progress: Scripture mastery levels and study list items

For guest users (no account), this data:

  • Stays on your device only
  • Is not transmitted to any server
  • Is not shared with any third party
  • Can be deleted by uninstalling the App or clearing app data

Account & Cloud Sync (Optional)

If you choose to create an account, we store additional information to enable cloud sync:

Account Information:

  • Email address (for email sign-up) or OAuth identifier (for Google/Apple sign-in)
  • Display name (optional)
  • Account creation date

Synced Progress Data:

When you're signed in, your game progress syncs to our cloud service to enable:

  • Cross-device sync (play on multiple devices)
  • Progress backup (restore if you get a new device)
  • Continuity (pick up where you left off)

Data we sync:

  • Game history and scores
  • XP, levels, and achievements
  • Streaks and daily goals progress
  • Scripture mastery progress
  • Missed questions for study review
  • Personal best records

Data we do NOT sync (stays local only):

  • Game preferences (sound, instructions shown)
  • Temporary game state

Your synced data is stored securely in our database (hosted by Supabase) and is associated with your account. You can delete all cloud data at any time by deleting your account.

Anonymous Analytics Data

To improve the App and fix bugs, we collect limited anonymous usage data:

Error Tracking (Sentry)

  • App crash reports
  • Error messages and stack traces
  • Device type and operating system version
  • App version

Usage Analytics (PostHog)

  • Screen views (which screens you visit)
  • Game events (game started, game completed, achievements unlocked)
  • Session duration
  • Device type and operating system version

What we DO NOT collect:

  • Personal identification information (name, email, phone) — except for optional accounts
  • Location data
  • Advertising identifiers
  • Your game progress, scores, or achievements (in analytics)
  • Content of your study lists
  • Any data that could identify you personally (analytics are anonymous)

All analytics data is:

  • Anonymous (no user IDs or personal identifiers)
  • Aggregated (we see trends, not individual users)
  • Used solely to improve the App

How We Use Information

Local Data

We use locally stored data solely to:

  • Track your progress across game sessions
  • Display your achievements and statistics
  • Maintain your daily streaks
  • Remember your preferences

Account Data

We use account and synced data solely to:

  • Authenticate you across devices
  • Sync your progress between devices
  • Restore your progress if you get a new device

Analytics Data

We use anonymous analytics data to:

  • Identify and fix bugs and crashes
  • Understand which features are most used
  • Improve app performance
  • Guide development of new features

Third-Party Services

The App uses the following third-party services:

Supabase (Account & Data Storage)

  • Purpose: User authentication and cloud data storage
  • Data collected: Account info, synced game progress (for signed-in users only)
  • Privacy policy: https://supabase.com/privacy
  • Note: Only used if you create an account; guest users have no data with Supabase

Sentry (Error Tracking)

  • Purpose: Crash reporting and error monitoring
  • Data collected: Crash reports, device info, app version
  • Privacy policy: https://sentry.io/privacy/

PostHog (Usage Analytics)

  • Purpose: Anonymous usage analytics
  • Data collected: Screen views, game events, session data
  • Privacy policy: https://posthog.com/privacy

Analytics services (Sentry, PostHog) receive only anonymous, non-personal data. Supabase receives account information only if you choose to create an account.

Your Choices

Opting Out of Analytics

  • Analytics are completely optional for app functionality
  • The App works fully offline without sending any data
  • Future versions may include an in-app toggle for analytics

Deleting Your Data

Local Data:

To delete data stored on your device:

  • iOS: Delete the app, or go to Settings > General > iPhone Storage > Book of Mormon Mastery > Delete App
  • Android: Delete the app, or go to Settings > Apps > Book of Mormon Mastery > Clear Data

Account & Cloud Data:

If you have an account and want to delete all cloud data:

  1. Open the App and go to Profile
  2. Tap "Delete Account"
  3. Confirm deletion

This will:

  • Permanently delete your account
  • Delete all synced progress data from our servers
  • Clear all local data on that device
  • This action cannot be undone

Analytics Data:

Analytics data is anonymous and cannot be traced back to you or deleted on request (because we don't know which data is yours).

Data Sharing

We do not sell any data.

The App:

  • Does not contain advertisements
  • Does not sell data to third parties
  • Does not share data for marketing purposes
  • Only shares anonymous data with analytics providers as described above

Data Security

Local Data: Stored using your device's standard secure storage mechanisms (iOS Keychain/Android Keystore-backed storage).

Account Data: Transmitted securely via HTTPS. Stored in Supabase's PostgreSQL database with Row Level Security (RLS) ensuring you can only access your own data. Authentication tokens stored securely using Expo SecureStore.

Analytics Data: Transmitted securely via HTTPS to Sentry and PostHog, who maintain industry-standard security practices.

Data Retention

Local Data: Retained on your device until you delete the app or clear app data.

Account Data:

  • Retained while your account is active
  • Deleted immediately upon account deletion request
  • We do not retain backups of deleted account data

Analytics Data:

  • Sentry: Error data retained for 90 days
  • PostHog: Usage data retained per PostHog's retention policy

Children's Privacy

The App is educational in nature and appropriate for users of all ages.

Guest users: No personal information is collected.

Account creation: Creating an account requires an email address or social sign-in. Users under 13 should have parental consent before creating an account. The App can be fully used without an account.

Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page and updating the "Last Updated" date.
